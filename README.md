# DuckLY-Local-Assistant
#### This is a self-hosted local assistant built for locally served Ollama servers.
# About this project:
This is an **open** `AI` project built using **pure** *Python*, and my task was to make it with***OUT*** using **ANY** external libraries... And it finally ***SUCCEED***!
I've wrote these *~1000* lines alone, I tested it, works well. Enables the ability to voice chat with the assistant, send messages, and much more!
# Features:
* Voice chat
* Text messages
* File/directory operations
* **Agentic** *capabilities*, which include analyzing files, repositories, and even creating and generating projects!
# What is this for:
#### I've made this project mostly for:
* Others to analyze it
* Commenting it
* Learning from it
* Running it and & use
# How to use:
1. Prepare your `Ollama` server with the model
2. Take and download the source code (`DuckLY-L2.5`) script and run it with `Python`. No requirements there, as it depends only on pure standard library.
3. Run the code.
4. Chat/talk to the assistant.
5. Share some feedback if needed.
> There is a ready-made built executable with `PyInstaller`. You can download it and run on your hardware. It's the same script, but in binary form (executable).
