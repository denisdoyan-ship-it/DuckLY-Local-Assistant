# DuckLY-Local-Assistant
##### This is a self-hosted local assistant built for locally served Ollama servers.
# About this project:
This is an open AI project built using pure Python, and my task was to make it without using ANY external libraries... And it finally succeed!
I've wrote these ~1000 lines alone, I tested it, works well. Enables the ability to voice chat with the assistant, send messages, and much more!
# Features:
* Voice chat
* Text messages
* File/directory operations
* Agentic capabilities, which include analyzing files, repositories, and even creating and generating projects!
